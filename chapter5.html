Kasus Nyata: Menentukan Status Sistem Energi Keterbaruan Eco Techno Leader

Eco Techno Leader perlu mengecek status sistem energi keterbaruan berdasarkan kapasitas dan efisiensi. Kita akan gunakan pernyataan if untuk ini.

Level Beginner: Pernyataan if Sederhana

Kita cek apakah kapasitas sistem cukup.

kapasitas_sistem = 20000  # dalam watt
kapasitas_minimum = 15000
if kapasitas_sistem >= kapasitas_minimum:
    print("Kapasitas sistem cukup untuk operasi!")
else:
    print("Kapasitas sistem kurang, perlu ditambah.")

Hasil di Command Prompt:

Kapasitas sistem cukup untuk operasi!

Do: Gunakan if untuk kondisi sederhana.

Don't: Jangan lupa tambah indentasi (spasi 4) di bawah if atau else.

Level Intermediate: Menggunakan elif

Kita cek efisiensi sistem dengan beberapa kategori.

efisiensi = 85  # dalam persen
if efisiensi >= 90:
    print("Efisiensi sangat baik!")
elif efisiensi >= 80:
    print("Efisiensi baik, tapi bisa ditingkatkan.")
else:
    print("Efisiensi rendah, perlu perbaikan.")

Hasil di Command Prompt:

Efisiensi baik, tapi bisa ditingkatkan.

Do: Gunakan elif untuk kondisi bertingkat.

Don't: Jangan pake banyak if terpisah untuk kondisi yang saling terkait, lebih baik pake elif.

Level Advanced: Kombinasi Kondisi dengan and/or

Kita kombinasikan kapasitas dan efisiensi untuk status lengkap.

kapasitas_sistem = 20000
efisiensi = 85
kapasitas_minimum = 15000
efisiensi_minimum = 80
if kapasitas_sistem >= kapasitas_minimum and efisiensi >= efisiensi_minimum:
    print("Sistem siap beroperasi dengan baik!")
elif kapasitas_sistem < kapasitas_minimum or efisiensi < efisiensi_minimum:
    print("Sistem perlu perbaikan, cek kapasitas atau efisiensi.")

Hasil di Command Prompt:

Sistem siap beroperasi dengan baik!

Do: Gunakan and atau or untuk gabungkan kondisi.

Don't: Jangan buat kondisi terlalu panjang tanpa tanda kurung, biar gak bingung.
