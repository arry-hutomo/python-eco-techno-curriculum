Defining Variables

Variabel itu kayak kotak buat nyimpan data. Kamu bisa kasih nama apa aja, tapi ada aturan yang harus diikutin:

  • Nama variabel bisa pake huruf, angka, atau underscore (_).
  • Gak boleh mulai dengan angka.
  • Jangan pake kata kunci Python (misalnya str, if, dll.).
  • Hindari nama yang mulai dengan underscore, soalnya itu biasanya punya makna khusus.

Contoh cara bikin variabel:

miles = 10  # type integer
first_name = 'John'  # type string
last_name = "Wick"  # type string
a, b, c = 1.5, 3, 'x'  # mendefinisikan 3 variabel sekaligus

Python merekomendasikan pake snake_case untuk nama variabel, bukan camelCase. Contoh:

max_permitted_value = 500  # snake_case (disarankan)
maxPermittedValue = 500  # camelCase (kurang disarankan)

Contoh nama variabel yang salah atau gak disarankan:

# 4you = 10  # salah, nama gak boleh mulai dengan angka
# valules = 100  # salah, gak boleh ada karakter spesial
str = 'Python'  # gak disarankan, str adalah kata kunci Python
_location = 'Europe'  # gak disarankan, mulai dengan underscore

Comments in Python

Komentar di Python dimulai dengan tanda # dan berlanjut sampai akhir baris. Komentar ini gak akan dibaca sama Python, cuma buat kasih catatan.

# Ini adalah komentar
# x = 1  # baris ini dikomentari, jadi diabaikan
a = 7  # mendefinisikan variabel a
my_str = 'Tanda # dalam string # bukan komentar'
